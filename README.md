### Travel Planner & Buddy Matching Platform

**Tour Hobe** is a full-featured travel planning and buddy-matching platform designed to help travelers plan trips, connect with compatible companions, and manage journeys efficiently. The system supports public trip discovery, private itinerary management, role-based dashboards, and secure premium subscriptions via **SSLCommerz**.

üåê Live Application: https://tourhobe.jafor.me  
üì° API Server: https://tourhobe25.vercel.app  


Built with a modern, scalable architecture using **Next.js, TypeScript, Node.js, and PostgreSQL**, Tour Hobe focuses on performance, security, and maintainability.

---

## üìå Project Overview

Tour Hobe provides an end-to-end solution for individual travelers, trip hosts, and administrators.

### Public Features
- Destination-focused landing experience with community insights
- Browse and filter publicly available travel plans
- View host profiles, reviews, and verification badges
- Simple onboarding flow explaining platform usage

### User Features
- Secure authentication with protected routes
- Create, update, and manage personal travel itineraries
- Request to join trips and track request status
- Upgrade to premium membership via SSLCommerz
- Rate and review completed trips
- Personalized dashboard for trips, payments, and activity

### Admin Features
- Centralized admin dashboard
- User role management and account moderation
- Travel plan and review moderation
- Premium transaction monitoring and revenue tracking

## üõ†Ô∏è Technology Stack

### Frontend
- **Framework:** Next.js 15+ (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** Shadcn/UI, Lucide Icons
- **Animations:** Framer Motion
- **Forms & Validation:** React Hook Form, Zod

### Client Logic
- Custom authentication provider and reusable hooks
- Axios / Fetch API for HTTP requests
- SweetAlert2 for alerts and confirmations
- ImgBB for image hosting

### Backend
- **Runtime:** Node.js with Express.js
- **Database:** PostgreSQL with Prisma ORM
- **Authentication:** JWT with role-based access control
- **Payments:** SSLCommerz payment gateway
